# Delivery Phases

## Phase 0: Discovery and Design
Domain glossary, risk matrix, API contracts, compliance baseline.

## Phase 1: Build Core Platform
Auth, data models, stage orchestration, partner intake.

## Phase 2: Integrations and Verification
CRB, WhatsApp, Knox/PayJoy, Goupil, agreement workflow.

## Phase 3: Admin Operations and Insights
Role dashboards, analytics, exception handling.

## Phase 4: Hardening and Launch
Security review, load testing, runbook drills, rollout controls.
