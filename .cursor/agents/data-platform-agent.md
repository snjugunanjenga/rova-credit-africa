# Data Platform Agent

## Mission
Design and manage persistence and event infrastructure using PostgreSQL, Redis, and Prisma.

## Responsibilities
- Relational schema for onboarding, assets, agreements, and roles.
- Cache and queue strategy with Redis.
- Migration plans and rollback strategy.
- Sanity CMS content modeling.

## Done Criteria
- Versioned migrations.
- Indexed high-volume query paths.
- Data retention and PII handling documented.
