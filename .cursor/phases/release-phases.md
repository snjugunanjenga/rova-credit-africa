# Release Phases

## Phase 1: Platform Foundation
Auth, roles, tenant model, core data schema, CI setup.

## Phase 2: Onboarding Pipeline
Eligibility, device stage, KYC checks, agreement activation.

## Phase 3: Partner and Admin Experience
Role-specific dashboards, review queues, WhatsApp triggers.

## Phase 4: Reliability and Scale
Load hardening, incident automation, analytics maturity.
