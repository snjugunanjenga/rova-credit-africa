# Phase 3 - Partner and Admin Experience Backlog

## Stories
- P3-S1: Partner registration and activation workflow UI.
- P3-S2: Partner application dashboard with stage progress visibility.
- P3-S3: Admin Owner queue with override/approval controls.
- P3-S4: Admin Developer tools for integration diagnostics.
- P3-S5: Admin Analyst funnel and risk dashboards.
- P3-S6: Admin Marketer messaging templates and campaign triggers.

## Definition of done
- Role-specific UI routes protected server- and client-side.
- Exception queues actionable with audit logs.
- Accessibility and mobile-first checks pass.
